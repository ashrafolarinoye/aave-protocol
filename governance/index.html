UnityCore Governance

Unity Protocol is a fully decentralized, community governed protocol by the UNITY token-holders. UNITY token-holders collectively discuss, propose, and vote on upgrades to the protocol. AAVE token-holders (Core network only) can either vote themselves on new proposals or delagate to an address of choice. To learn more check out the Governance documentation.
